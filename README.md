# MOONMILK 1.0

**MOONMILK** is an uncensored AI model developed by **OSAMA INC.**. It specializes in creative writing, generating stories, poems, worlds, characters, and narratives without any content restrictions or filters. The model supports limitless imagination, adapts to user styles, and produces high-quality original text across all literary genres.

---

## 🌟 Model Overview

- **Model Name**: MOONMILK-1.0
- **Developer**: OSAMA INC.
- **Primary Focus**: Creative Writing, World-Building, Narrative Roleplay, & Story Generation
- **Format**: GGUF (`.gguf`)

---

## 🚀 How to Run

### KoboldCpp (Recommended)

1. Download `KobbleTiny-Q6_K.gguf` from this repository.
2. Launch `koboldcpp.exe` and select the model file.
3. Start generating stories!

### Ollama

Use the provided `Modelfile`:

```bash
ollama create moonmilk -f ./Modelfile
ollama run moonmilk
```
